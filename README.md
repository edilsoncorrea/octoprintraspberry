Tutorial sobre como instalar o Octoprint no Raspberry

Tutorial da comunidade do Octoprint bem detalhado sobre como instalar no Raspbian. Este tutorial discorre sobre duas formas de instalar. A primeira refere-se sobre a instalação utilizando uma imagem a ser colocada num cartão SD. A segunda é uma instalação a partir de um Linux, com criação do ambiente virtual, instalação do Python e usando npm:
[Setting up OctoPrint on a Raspberry Pi running Raspberry Pi OS (Debian)](https://community.octoprint.org/t/setting-up-octoprint-on-a-raspberry-pi-running-raspberry-pi-os-debian/2337)
